Flow Scoped - Page 1

Simple flow starts implicitly

A simple flow has an empty configuration files, nothing much more... The flow is derived implicitly.

  • simple-flow-flow.xml: By convention -flow.xml configuration file
  • simple-flow.xhtml: First page by conventions
  • simple-flow-return.xhtml: Return page by conventions
  • No managed bean is necessary: only flow-scoped data
  • Navigate from first page to other and use flowScope data
  • Navigate to return page to exit the flow. flowScope parameter is not usable any more.
