Decorator

Demonstrate the use of a decorator.

  • DecoratorBean: Managed bean for this page.
  • Worker: Interface to do some work.
  • WorkerImpl: The implementation class to do the work.
  • WorkerDecorator: Abstract class that will decorate the some work in WorkerImpl.
  • beans.xml: Specify the decorator.
  • The DecoratorBean and the WorkerImpl have also a logged interceptor. See result in the logfile.
