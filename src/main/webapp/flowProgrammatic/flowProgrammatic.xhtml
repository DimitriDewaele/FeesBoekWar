Page 1: Programmatic flow

The configuration of this flow is done in code.

  • Configuration Class must have the same name as the flow: FlowProgrammatic.java
  • Single method to define the flow: defineFlow

If you called this flow from the Configuration flow, then you can see these parameters:

  • flowScope.paramFromConfigurationValue1 = #{flowScope.paramFromConfigurationValue1}
  • flowScope.paramFromConfigurationValue2 = #{flowScope.paramFromConfigurationValue2}
